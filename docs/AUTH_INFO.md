# Admin Giriş Bilgileri

Admin paneli giriş sistemi aktif edilmiştir.

## Giriş Ekranı
Adres: `/admin` veya `/admin/login`

## Varsayılan Bilgiler
- **Kullanıcı Adı:** `admin`
- **Şifre:** `admin123`

## Güvenlik
- `/admin` altındaki tüm sayfalar korumalıdır.
- Oturum süresi 1 gündür.
- Çıkış yapmak için Sidebar altındaki butonu kullanabilirsiniz.
